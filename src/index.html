Дизайн

Modeler
Сделано для разработчиков, редактируйте диаграммы процессов BPMN и таблицы решений DMN в Camunda Modeler. Разверните свои рабочие процессы в Camunda Engines.
Cawemo
Сотрудничайте с несколькими заинтересованными сторонами на удобной для бизнеса платформе Cawemo, чтобы создавать, редактировать и указывать схемы процессов BPMN для развертывания вашими разработчиками.

Автоматизация

Cockpit
С помощью Camunda Cockpit вы можете отслеживать рабочие процессы и решения в производстве, чтобы обнаруживать, анализировать и решать технические проблемы. Cockpit - идеальный инструмент для выполнения технологических операций.
Tasklist
Список задач Camunda - это готовое к использованию веб-приложение, которое позволяет конечным пользователям работать над возложенными на них задачами. Он обеспечивает дополнительную наглядность при использовании Camunda Workflow Engine для управления человеческими задачами.
BPMN Workflow Engine
Camunda Workflow Engine - правильный выбор как для делегирования (микро) сервисов, так и для управления человеческими задачами. Вы можете использовать его как удаленную службу REST или встроить в свое приложение Java.
DMN Decision Engine
Camunda Decision Engine выполняет бизнес-ориентированные таблицы решений. Этот механизм предварительно интегрирован с механизмом рабочего процесса, но вы также можете использовать его как автономное приложение через REST или внутри вашего Java-приложения.

Улучшение

Optimize
Используйте Camunda Optimize, чтобы создавать красивые отчеты и размещать их на панели управления для бизнес-мониторинга. Настройте оповещения, которые срабатывают при пропуске целей по производительности, и используйте расширенную аналитику для выявления узких мест в процессе.
